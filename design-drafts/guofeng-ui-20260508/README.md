# 问云派古风国风 UI 设计稿索引

## 设计稿说明

本目录用于保存 2026-05-08 生成的古风国风 UI 页面设计总稿。设计稿由内置生图工具生成，目的是作为后续页面重构和视觉细化的参考稿，不直接参与网站运行。

整体方向：宣纸底纹、青绿山水、朱砂印章、窄金线、宋体标题、轻量卡片、清晰表单、稳定后台工作台。

## 页面结构梳理

### 前台展示页

1. 首页：山门首屏、五个快捷入口、核心承诺、门派精神、流程、云灯、公告。
2. 立派金典：章节目录、长文阅读、返回顶部。
3. 山门介绍：立派愿景、门风、宗门架构。
4. 门派公告：公告列表、置顶标识、发布时间。
5. 问云雅集：活动卡、时间地点、报名状态。
6. 联系山门：联系说明、二维码安全、登记入口。
7. 404 页面：空路提示、返回首页。

设计稿路径：`01-public/public-display-pages-guofeng-ui.png`

2. 问心考核：题目进度、答题卡、选项、交卷操作。
3. 云灯留言：点灯表单、匿名开关、留言列表。
4. 问云小院登录：登录、注册、找回密码、重置密码入口。

设计稿路径：`02-interaction/public-interaction-pages-guofeng-ui.png`

3. 我的名帖：名帖状态、审核进度、资料修改提示。
4. 我的云灯：本人云灯列表、审核状态。
5. 我的雅集：报名记录、取消报名、活动状态。
6. 消息提醒：通知列表、已读未读、时间线。

设计稿路径：`03-yard/yard-pages-guofeng-ui.png`

### 后台管理页面

1. 后台总览：待审核登记、待审核云灯、公告数量、活动数量。
2. 名册管理：搜索筛选、展开审核、问心分数、归云堂状态。
3. 云灯审核：留言列表、通过、拒绝。
4. 公告管理：创建公告、草稿、发布、公告列表。
5. 活动管理：创建活动、报名名单、线上线下状态。
6. 执事管理：成员列表、角色授权、邮箱和密码维护。
7. 站点设置：联系山门、归云堂二维码、邮件服务设置。

## 生图提示词摘要

### 前台展示页

生成一张横版高清网站 UI 设计总稿，覆盖首页、立派金典、山门介绍、门派公告、问云雅集、联系山门、404 页面。风格为高级古风国风，使用宣纸底纹、淡墨远山、青绿山水、朱砂印章、窄金线、宋体标题、留白和小圆角卡片。

### 前台互动页

生成一张横版高清网站 UI 设计总稿，覆盖问云名册、问心考核、云灯留言、问云小院登录页面。重点展示表单、答题卡、筛选、留言墙、登录面板和提示条，风格温暖但有边界。

### 问云小院页面

生成一张横版高清用户后台 UI 设计总稿，覆盖小院总览、我的资料、我的名帖、我的云灯、我的雅集、消息提醒。重点展示浅色书册工作台、固定侧栏、统计卡、资料表单和通知列表。

### 后台管理页面

生成一张横版高清管理后台 UI 设计总稿，覆盖后台总览、名册管理、云灯审核、公告管理、活动管理、执事管理、站点设置。重点展示深青色执事堂侧栏、宣纸内容区、表格、筛选器、状态标签和危险操作分层。

## 后续落地建议

1. 先统一全站视觉变量：背景、主色、警示色、金线、卡片阴影、标题字号。
2. 再改公共组件：顶部导航、侧栏、`SectionTitle`、`ScrollPanel`、`RitualCard`、表单、表格、按钮。
3. 最后按页面分批落地：前台展示页、前台互动页、小院页面、后台页面。
4. 每批落地后都需要跑构建，并检查桌面端和移动端首屏。
